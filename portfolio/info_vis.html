Information Visualization

"Information visualization is the study of (interactive) visual representations of abstract data to reinforce human cognition. The abstract data include both numerical and non-numerical data, such as text and geographic information."


The amount of information that needs to be processed in a clear and effective manner each day is far too much for a person to do manually. With the help of interactive and dynamic visual design structures, information visualization is the key to data analysis and provides the possibility to extract the information the user can need. I have specialized in data analysis - the ability to understand what information a user might want and how to extrat and visualize this data in the most effective way possible. As with any project, a lot of these can be further expanded and enhanced! At times I will go back and make changes and additions as I see fit in order to make them even more complete!

MT Map (WIP)
The goal of MT Map is so that a user can plan a vacation or their bucket list all in one place. The user can track their 'world progress' and places they've been, record notes on what they did in each country and if they would go back. It's like a visual interactive travel diary!

HGC Esports Analysis Tool

Food Advisor

Visualizing World Data
Hans Rosling was the grandfather of Visualizing data and this project is dedicated to him. With the use of parallel coordinates along with a wide variety of visualization techniques you can easily visualize and find relationships between specific statistics.

Group Creator
This was my first ever made information visualization project. The aim was to take a big data set of people with character traits and their own comments about personal strengths and interest, visualize these traits and create a tool that allows a moderator to effectively sort the people into 10 groups. The visualization transforms the dataset into something more cognitive by using a radar chart and plenty of colors.
