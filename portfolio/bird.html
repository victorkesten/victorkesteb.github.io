Bird

Bird was a flight 'simulation' game created using the Unity Game Engine. The goal of the project was to combine interactive means of controls and advanced graphics and ultimately creating an interesting product. Our team decided to use the highly underrated and latest version of Microsoft Kinect as it allows for a human to control to cocntrol things on the screen without using a handeheld device.
