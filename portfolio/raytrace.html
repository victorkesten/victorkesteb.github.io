Why a ray tracer?

A ray tracer is one of the most elementary techniques in computer graphics. This technique is used for generating an image by tracing the path of light through pixels in an image plane and simulating the effects of its encounters with virtual objects. Together with the Cornell Cubes, I created a Ray Tracer that can accurately trace rays and generate beautiful imagery. The code is set up using SDL and accurately implements linear interpolation techniques along with simple shadow tracing as well as reflection, refraction and scattering.
