Current Projects:

2018 Swedish General Election

Our Swedish general election happened on September 9th and the government makes all voter data available to the public. This data is however fairly difficult to navigate in an effective way! I, being a problem solver, am in the process of creating a visualization tool that organizes the data and allows users to browse it more easily with the goal of spotting trends and patterns!



SKL

In my professional life I'm currently working as a freelance data analyst for Sveriges Kommuner och Landsting (government section dedicated to state organisation) on a project called #skolDigiPlan. My role in this project is to measure and visualize user engagement in this plan, elucidate discussion results based on pre-determined questions and to provide feedback on the discussion platform they use (called VoteIT). Hopefully soon I will be able to share some visualizations based on the data collected!



Garmin Running Data

Long distance running is my main hobby right now and in June 2018 I ran my first Marathon (and the second on September 1st, 2018). Running a marathon requires a lot of prepartion and practice - something that can be tracked! With the use of my new Garmin Forerunner 235 tempo watch I'm able to extract each activity session and collect a ton of super interesting data! Seeings as one of my favourite activities is to visualize data, the only logical thing to do next was to create a cool interactive tool which I can use on my journey!
