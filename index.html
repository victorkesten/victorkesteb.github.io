Current Projects:

SKL

In my professional life I'm currently working as a freelance visualization specialist for Sveriges Kommuner och Landsting (Swedish association of local authorities and regions) on a project called #skolDigiPlan. My role in this project is to measure and to create a visualization tool for user engagement, to elucidate discussion results based on pre-determined questions from their forum like tool and to provide feedback on the discussion platform they use (called VoteIT). My second responsibility is to create a visualization tool for in support of the appropriate users can create their own strategic action plans when implementing digitization methods within a school setting.

Garmin Running Data

Long distance running is my main hobby right now and in June 2018 I ran my first Marathon (and the second on September 1st, 2018). Running a marathon requires a lot of preparation and practice - something that can be tracked! With the use of my new Garmin Forerunner 235 tempo watch I'm able to extract each activity session and collect a ton of super interesting data! Seeing as one of my favourite activities is to visualize data, the only logical thing to do next was to create a cool interactive tool which I can use on my journey!
